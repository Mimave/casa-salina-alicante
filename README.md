# Casa Salina Alicante

Proyecto de clase S3 · Hotel & Turismo. Web React + TypeScript + Vite lista para Vercel.

## Arrancar en Codex o terminal

```bash
npm install
npm run dev
```

Abre la URL local que muestra Vite, normalmente `http://localhost:5173`.

## Comprobar antes de entregar

```bash
npm run build
npm run preview
```

## Publicar en Vercel

### Opción 1 · Desde GitHub
1. Sube esta carpeta a un repositorio.
2. En Vercel pulsa **Add New → Project**.
3. Importa el repositorio.
4. Vercel detectará Vite automáticamente.
5. Build command: `npm run build`.
6. Output directory: `dist`.
7. Pulsa **Deploy**.

### Opción 2 · Vercel CLI

```bash
npm i -g vercel
vercel
```

Responde las preguntas, acepta la configuración detectada y luego ejecuta:

```bash
vercel --prod
```
