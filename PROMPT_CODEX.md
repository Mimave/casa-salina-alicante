# Prompt maestro para Codex

Trabaja como frontend lead y diseñador de producto senior. Revisa este proyecto React + TypeScript + Vite para Casa Salina Alicante. Respeta `Brand_Book.md` y `PRD.md` como fuentes de verdad.

Objetivos:
1. Mantener una estética mediterránea premium, serena y mobile first.
2. No eliminar ninguna sección o funcionalidad existente.
3. Corregir cualquier error de TypeScript, accesibilidad, responsive o build.
4. Mantener el cálculo de noches/precio, lightbox, FAQs, menú móvil y toast.
5. No introducir backend, Stripe ni secretos en S3.
6. Ejecutar `npm run build` antes de terminar.
7. Explicar en un resumen final qué se cambió y cómo publicarlo en Vercel.

No sustituyas el diseño por una plantilla genérica. Prioriza precisión, rendimiento, contraste, jerarquía visual y una experiencia memorable.
